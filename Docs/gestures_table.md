# Таблица жестов для взаимодействия с VR-средой


## 1. Телепортация

| Механика | Жест (реальный) | Эмуляция (клавиша) |
|----------|----------------|--------------------|
| Активация режима телепортации | Указательный палец вытянут, остальные сжаты («пистолет») | Зажать **F2** |
| Прицеливание | Вектор от кончика указательного пальца. На земле отображается точка/круг | Автоматически при зажатом F2 (луч из камеры) |
| Подтверждение телепортации | Щипок большим и указательным (имитация нажатия) | Нажать **F1** (не отпуская F2) |
| Телепортация | Мгновенное перемещение в точку прицела | Отпустить F1 (оставаясь в режиме F2) или сразу оба |

> **В эмуляции:** зажали F2 → появился луч. Нажали F1 (не отпуская F2) → телепорт. Отпустили F1 → можно снова телепортироваться.

---

## 2. Непрерывное движение

| Механика | Жест (реальный) | Эмуляция (клавиша) |
|----------|----------------|--------------------|
| Активация движения | Раскрытая ладонь пальцами вниз (горизонтально) | Зажать **F3** |
| Регулировка скорости | Чем ниже ладонь к полу — тем быстрее | Скорость фиксирована (1 м/с) или регулируется колёсиком мыши (зажатый F3 + колёсико) |
| Направление | По направлению руки (в эмуляции — куда смотрит камера) | Автоматически вперёд по взгляду |
| Остановка | Поднять ладонь вверх или сжать пальцы | Отпустить F3 |

> **В эмуляции:** зажали F3 → движение вперёд. Отпустили → остановка.

---

## 3. Захват объекта

| Механика | Жест (реальный) | Эмуляция (клавиша) |
|----------|----------------|--------------------|
| Захват | Щипок (большой + указательный) в момент касания объекта | Нажать **F1** (когда объект рядом с рукой) |
| Удержание | Пальцы сжаты, объект приклеен к руке | Держать F1 зажатой |
| Отпускание | Разжать пальцы | Отпустить F1 |

> **В эмуляции:** F1 — это «щипок». Зажали → схватили ближайший объект. Отпустили → бросили.

---

## 4. Вызов контекстного меню

| Механика | Жест (реальный) | Эмуляция (клавиша) |
|----------|----------------|--------------------|
| Вызов меню | Двойной хлопок руками перед собой | Нажать **F4** |
| Закрытие меню | Разжать пальцы или повторить жест | Повторно нажать F4 |
| Навигация по меню | Указательный жест правой руки («тычок») | Двигать мышь (курсор появляется на меню) |
| Выбор пункта | Щипок (большой + указательный правой руки) | **Левый клик** мыши |

> **В эмуляции:** F4 — открыть/закрыть меню. Курсором мыши навести на кнопку, левый клик — нажать.

---

## Сводная таблица клавиш (для разработчиков)

| Клавиша | Что делает в эмуляции |
|---------|----------------------|
| **F1**  | Щипок (захват / отпускание / подтверждение телепортации) |
| **F2**  | Указательный жест (включить режим телепортации, луч) |
| **F3**  | Открытая ладонь (непрерывное движение вперёд) |
| **F4**  | Вызвать / закрыть системное меню |
| **Мышь** | Вращение камеры, курсор для меню |
| **Левый клик** (при активном меню) | Нажатие кнопки в меню |

---

## Примечания

- В эмуляции **не поддерживается** регулировка скорости непрерывного движения (используется фиксированная скорость).
- Режим телепортации и режим непрерывного движения не должны быть активны одновременно (в коде можно добавить блокировку).
- При реальном трекинге (после получения VR-очков) клавиши заменятся на распознавание жестов через OpenXR.
